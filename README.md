# equipoArq - Sitio Web Institucional

Este repositorio contiene el sitio web institucional de equipoArq, un estudio orientado a arquitectura y construccion.

El proyecto fue desarrollado para comunicar de forma clara la identidad del estudio, sus servicios, trabajos ejecutados y canales de contacto. Su objetivo principal es representar la propuesta profesional de la empresa en entorno digital, con foco en la presentacion de contenidos y la experiencia de navegacion.

## Alcance del proyecto

- Presentacion institucional de equipoArq.
- Exposicion de servicios para clientes y profesionales.
- Muestra de trabajos y antecedentes.
- Formulario de contacto para consultas.

## Motivo de incorporacion a Git

Aunque el sitio ya existia previamente y no contaba con repositorio, se decidio incorporarlo a Git para ordenar su evolucion y administrar sus cambios de manera formal.

Esta decision permite:

- Mantener trazabilidad de modificaciones en codigo y contenidos.
- Mejorar el resguardo historico del proyecto.
- Facilitar el trabajo de mantenimiento y continuidad en el tiempo.
- Reducir riesgos ante cambios no documentados.

## Nota de confidencialidad

Este repositorio corresponde a un activo de empresa. Por ese motivo, la documentacion se limita a una descripcion general del proyecto y del proceso de versionado, sin incluir detalles operativos internos.
